# Water-Model---Agricultural-Field
Spatially explicit Python model for agricultural water accounting (blue, green &amp; grey water) following WFLDB v3.10 &amp; ISO 14046. Computes all 5 water output flows as GeoTIFF rasters using Pfister et al. (2011) ETirr, ICID 2021 irrigation efficiency, and Siebert et al. (2010) water origin data.
